# Health Score Guide — Client Success Strategist

## Dimensão 1 — Resultado (0–10)

Avalia se o cliente percebe entrega de valor real. A pontuação ancora no **sinal primário** — o que o cliente disse sobre resultados e ROI. Métricas de campanha são fonte secundária: ajustam o score mas não definem a leitura.

| Faixa | Critério |
|-------|----------|
| 9–10 | Cliente expressou satisfação explícita com resultados. Metas sendo superadas ou alinhadas à expectativa declarada. Referência positiva a ROI ou impacto no negócio. |
| 7–8 | Resultados dentro do esperado sem reclamação ativa. Cliente não questionou valor. Métricas secundárias saudáveis. |
| 5–6 | Cliente questionou uma meta específica ou comparou performance com expectativa original. Sem crise mas com ceticismo pontual. |
| 3–4 | Cliente expressou frustração com resultados ou questionou o valor da parceria. Resultado percebido abaixo do contratado. |
| 0–2 | Cliente sinalizou falha grave de resultado: solicitou revisão contratual, mencionou término, ou negou valor explicitamente. |

**Armadilha comum:** bom ROAS interno com cliente insatisfeito = Resultado baixo. O score reflete percepção do cliente, não métrica da plataforma.

---

## Dimensão 2 — Operação (0–10)

Avalia se a operação está funcionando do ponto de vista do cliente e internamente. Combina sinal primário (como o cliente percebe a operação) com evento secundário (erros, atrasos, disfunções identificadas).

| Faixa | Critério |
|-------|----------|
| 9–10 | Operação invisível para o cliente — flui sem fricção. Time executando sem bloqueios. Nenhum incidente operacional recente. |
| 7–8 | Operação funcional com pequenas fricções. Nenhum incidente crítico. Cliente não sinalizou problema de processo. |
| 5–6 | Um ou dois incidentes operacionais resolvidos mas que geraram fricção visível. Cliente notou mas aceitou a resolução. |
| 3–4 | Incidentes recorrentes afetando a experiência do cliente. Cliente sinalizou insatisfação com processo, comunicação ou execução. |
| 0–2 | Colapso operacional visível: erros graves com impacto direto no cliente, falha de comunicação sistemática, cliente ativamente frustrado com o processo. |

---

## Dimensão 3 — Entregas & Qualidade (0–10)

Avalia se o que foi entregue chegou no prazo e com a qualidade percebida como adequada pelo cliente.

| Faixa | Critério |
|-------|----------|
| 9–10 | Entregas no prazo, cliente validou qualidade sem ressalvas, feedback positivo explícito sobre algum output. |
| 7–8 | Entregas cumpridas com pequenas revisões. Qualidade aceitável sem crítica substantiva. |
| 5–6 | Atrasos pontuais ou revisões acima do esperado pelo cliente. Qualidade questionada em algo específico. |
| 3–4 | Padrão de atraso ou qualidade abaixo da expectativa declarada do cliente. Retrabalho frequente. |
| 0–2 | Entrega crítica não realizada, qualidade rejeitada pelo cliente, ou falha que gerou impacto no negócio do cliente. |

---

## Dimensão 4 — Relacionamento (0–10)

Avalia a qualidade do vínculo. É a dimensão mais difícil de pontuar sem sinal primário — e a mais estratégica para prever retenção e expansão.

| Faixa | Critério |
|-------|----------|
| 9–10 | Cliente é parceiro ativo: compartilha contexto estratégico, introduz o time a outros stakeholders, pede conselho além do escopo. Comunicação fluida e confiante. |
| 7–8 | Relacionamento sólido. Cliente engajado, responde rapidamente, comparece às reuniões. Confiança estabelecida. |
| 5–6 | Relacionamento funcional mas transacional. Cliente participativo apenas no necessário. Sem sinal de expansão de vínculo. |
| 3–4 | Engajamento reduzido: difícil de agendar, respostas lentas, cliente desviou para o operacional sem compartilhar estratégia. |
| 0–2 | Relacionamento deteriorado: cliente confrontacional, escalou para liderança, ignorou comunicações, ou sinalizou mudança de fornecedor. |

**Regra especial:** mudança de stakeholder (troca do contato do cliente) é automaticamente sinal de alerta para Relacionamento — marque como fonte a investigar mesmo sem outro sinal primário.

---

## Score Geral e Clima

O score geral é a **média das 4 dimensões** (não ponderada — cada dimensão tem o mesmo peso na média).

```
Score Geral = (Resultado + Operação + Entregas&Qualidade + Relacionamento) / 4
```

**Regra de clima:** a pior dimensão tem poder de veto.
- Se qualquer dimensão ≤ 4, o clima é no mínimo 🔴 Crítico — mesmo que a média seja alta
- Se qualquer dimensão = 5–6, o clima é no mínimo 🟡 Atenção
- 🟢 Saudável exige **todas** as dimensões ≥ 7

**Por que a pior dimensão tem poder de veto:**
Um relacionamento de 9 com resultado de 3 é uma conta em risco — não uma conta média de 6. O cliente feliz com o relacionamento mas frustrado com resultado não renova. O score geral informa, mas o clima é definido pelo elo mais fraco.
