# Entry Protocol — Decision Recorder

## Filtro de Entrada

Antes de registrar qualquer item, responda estas perguntas:

| Pergunta | Se "não" → |
|---|---|
| Isso muda uma decisão futura? | Descartar |
| Isso evita um erro futuro? | Descartar |
| Isso melhora execução ou qualidade? | Descartar |
| Isso ajuda a interpretar melhor a conta? | Descartar |
| Isso é recorrente ou estrutural (não pontual)? | Manter como hipótese apenas |
| Isso tem evidência suficiente? | Registrar como hipótese — não como truth |

**Regra prática:** se nenhuma das perguntas retornar "sim", não registrar.

---

## Hierarquia de Evidência

### Alta (→ Truth ou Trap)
- Fala explícita do cliente
- Decisão formal documentada
- Dado quantitativo confiável
- Comportamento recorrente do cliente (≥ 3 ocorrências)
- Padrão consistente observado ao longo do tempo
- Histórico validado

### Média (→ Hipótese ou Trap provisória)
- Evento operacional relevante único
- Comportamento do cliente interpretado (não confirmado)
- Revisão interna consistente com histórico
- Padrão observado 1–2 vezes

### Baixa (→ Hipótese apenas — nunca Truth)
- Feeling isolado de uma pessoa
- Percepção interna sem validação externa
- Leitura estratégica ainda aberta
- Suspeita sem dado de suporte

---

## Tipos de Registro e Critérios

| Tipo | Critério de entrada | Evidência mínima |
|---|---|---|
| **Truth** | Fato validado por dado, cliente ou operação consistente | Alta |
| **Trap** | Armadilha conhecida — já gerou fricção ou erro | Alta ou Média |
| **Hipótese** | Leitura relevante ainda não validada | Média ou Baixa |
| **Decisão** | Escolha estratégica tomada com racional documentado | Alta |
| **Sensibilidade** | Preferência, dor ou limite confirmado do cliente | Alta ou Média |

---

## Origens e Peso

| Origem | Hierarquia | Entra como |
|---|---|---|
| Sinal direto do cliente | Primária | Truth / Trap / Sensibilidade |
| Dado confiável (performance, métricas) | Primária | Truth |
| Evento operacional (entrega, erro, resultado) | Secundária | Trap / Hipótese |
| Revisão interna consistente | Secundária | Hipótese → Truth com confirmação |
| Percepção interna isolada | Terciária | Hipótese apenas |

**Regra:** fonte terciária nunca define clima, verdade ou armadilha sem confirmação externa.
